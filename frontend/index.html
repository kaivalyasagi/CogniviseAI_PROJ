About & API
Platform documentation, architecture overview, and API reference

What is Cognivise?

An AI-driven cognitive assessment platform powered by Google Gemini that analyses how students think — not just whether answers are correct. It builds a profile across reasoning patterns, concept gaps, and confidence.

  • Open-ended question generation per topic
  • Per-answer response capture (no MCQs)
  • Reasoning pattern detection
  • Concept gap & confusion analysis
  • Clarity + confidence scoring
  • Learning trend tracking over time
  • Per-user cognitive profile engine

Analytics Engine (6 Modules)

All session data is stored in data/sessions.json and processed by:

  • scoring_engine — calculate_score(first, final, correct)
  • topic_analysis — topic_statistics() + average_scores()
  • thinking_patterns — thinking_distribution()
  • error_analysis — error_percentage() per confusion type
  • trend_analysis — learning_trend() per user
  • cognitive_profile — dominant pattern + common confusion

Live API Routes

Score Formula

score = (final_attempt / correct_answer) × 70
      + max(improvement, 0) × 3
where improvement = final_attempt − first_attempt

Rewards both accuracy and measurable growth between first and final attempt.
