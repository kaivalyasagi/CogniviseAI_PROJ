# Cognivise AI – Demo Flow

This document describes how the Cognivise AI system will be demonstrated.

## Step 1 – User Opens the Application
The user opens the web application hosted on AWS S3 through their browser.

## Step 2 – Select a Concept
The user selects a concept they want to learn (example: Recursion, Binary Trees, Sorting).

## Step 3 – Diagnostic Question Generation
The system generates a set of diagnostic questions to analyze the user's understanding.

## Step 4 – User Submits Answer
The user answers the diagnostic question through the interface.

## Step 5 – Confusion Analysis
The backend processes the response and the AI engine identifies the type of conceptual confusion.

## Step 6 – Mental Model Repair
The system provides a structured explanation that corrects the misunderstanding.

## Step 7 – Reinforcement Task
The system generates a small exercise to reinforce the corrected concept.

## Step 8 – Dashboard Insight
The dashboard shows:
- Confusion type detected
- Concepts the user struggles with
- Learning progress over time
